# Systems Programming & Performance Engineering Portfolio

A high-performance, minimalist portfolio showcasing specialized work in **Systems Engineering**, **Machine Learning**, and **Full-stack Development**.

- **Virtual Filesystem Terminal**: An interactive retro-style terminal for exploring developer bio and projects (`ls`, `cd`, `cat`).

- **Unified Blog Aggregation**: Automated RSS parsing from **Substack**, **Medium**, and **TheCoreDump**.
